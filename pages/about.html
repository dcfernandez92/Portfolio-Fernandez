Sobre mí

Hola! Mi nombre es Diego y disfruto creando cosas para la web y aplicaciones celulares. Mi interés en el desarrollo web, comenzó en 2016 cuando intenté crear mi primer página web. Esto derivó a inscribirme en la Universidad Tecnológica Nacional y allí fue donde nació mi fanatismo por el mundo de la programación.

Luego de 3 años de estudios tuve el privilegio de dedicarme como desarrollador Backend para Nación Servicios, empresa en la cual continuó trabajando y adquiriendo nuevos conocimientos. Actualmente, me dedico exclusivamente al desarrollo de microservicios para la red de transporte SUBE.

Adicionalmente, comencé un curso de desarrollo fullstack para fortalecer el conocimiento con el uso de nuevas tecnologías.

A continuación mencionó algunas de las tecnologías que he utilizado recientemente:

  • Java
  • SpringBoot
  • C#
  • JavaScript
  • Node.js
  • React
